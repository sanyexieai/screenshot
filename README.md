# 项目名称

这是一个用Rust编写的截图工具(开发中)。

## 功能

- 截取屏幕截图
- 保存截图为文件
- 支持多种图片格式
- 支持多屏幕
- 支持多平台
- 支持固定图片到桌面

## 安装

请确保已安装Rust环境。然后运行以下命令：

```bash
git clone https://github.com/sanyexieai/screenshot.git
cd screenshot
cargo build --release
```

## 使用

运行以下命令启动截图工具：

```bash
cargo run
```
